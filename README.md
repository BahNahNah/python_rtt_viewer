# python_rtt_viewer
basic rtt viewer for NRF5X devices.

It has a few avalible actions

|Key  | Action        |
|-----|-----------    |
|c    | Clear window  |
|t    | Timestamp on/off|
|s    | Stopwatch start/stop|


### naming devices
`devices.json` can be created to associate names to serial numbers

Example:

`
{
    "1234567": "Dev_board_1"
}
`
